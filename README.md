# Тычка

Проект «Тычка» — современная чат-рулетка с видео, микрофоном и текстовым чатом.

## Что реализовано

- Вход / регистрация на фронтенде
- Поиск случайного собеседника
- Видео и аудио подключение
- Текстовый чат
- Админ-панель с банами и жалобами
- Сервер на Express + Socket.IO

## Запуск

1. Установить зависимости:
   ```bash
   npm install
   ```
2. Запустить сервер:
   ```bash
   npm run server
   ```
3. В другом окне терминала запустить фронтенд:
   ```bash
   npm run dev
   ```

Фронтенд доступен на `http://localhost:4173`.
Сервер Socket.IO доступен на `http://localhost:4174`.
